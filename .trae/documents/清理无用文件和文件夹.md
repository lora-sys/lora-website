## 可安全删除的文件和文件夹

### 1. `.next/` (构建缓存目录)
- 可完全删除，下次运行 `bun dev` 或 `bun build` 会自动重新生成
- 体积巨大，节省大量空间

### 2. `skills/__MACOSX/` (macOS 系统文件)
- macOS 自动生成的元数据文件夹
- 完全无用，包含大量以 `._` 开头的隐藏文件

### 3. `build_full.log` (构建日志)
- 历史构建日志
- 保留价值低

### 4. `public/sw.js` 和 `public/workbox-*.js` (PWA 文件)
- 由 `next-pwa` 自动生成
- 会在构建时自动重新生成

---

## 删除命令

```bash
# 删除构建缓存
Remove-Item -Recurse -Force .next

# 删除 macOS 系统文件
Remove-Item -Recurse -Force skills/__MACOSX

# 删除构建日志
Remove-Item build_full.log

# 删除 PWA 生成文件
Remove-Item public/sw.js
Remove-Item public/workbox-*.js
```
